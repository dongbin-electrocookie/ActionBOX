ActionBOX 사용자 매뉴얼

ActionBOX는 단축키, 매크로, 프로그램 실행, 미디어 제어, 마우스 제어를 버튼과 조이스틱, 로터리 엔코더에 배치해 사용하는 컨트롤러입니다.

PC에서는 PC Sync 모드로 프로그램과 실시간 연동하고, 모바일 기기에서는 오프라인 모드로 저장된 설정을 단독 실행할 수 있습니다.

PC Sync 오프라인 저장 조이스틱 아날로그 플러그인

처음 연결하기

  1. ActionBOX 본체를 PC에 USB로 연결합니다.
  2. ActionBOX.exe를 실행합니다. 프로그램 설치 파일이 없다면 다운로드 링크가 준비된 후 최신 버전을 받습니다.
  3. 상단의 연결 상태가 PC Sync ON 또는 연결된 기기 상태로 표시되는지 확인합니다.
  4. 왼쪽 기기 화면에서 원하는 버튼을 클릭한 뒤, 오른쪽 기능 목록에서 사용할 기능을 드래그합니다.
기본 흐름: 버튼 선택 → 기능 드래그 → 아래 설정 패널에서 옵션 조정 → 기기에서 테스트 순서로 사용하면 됩니다.

2. 하드웨어

ActionBOX는 10개의 메인 키, 조이스틱, 로터리 엔코더, 사이드 키, 디스플레이와 RGB LED로 구성됩니다.

구성 요소 역할
메인 키 F15-F24 단축키, 실행, 플러그인 기능, 이미지, 애니메이션, LED 설정을 배치하는 기본 버튼입니다.
조이스틱 4방향, 8방향, 아날로그 모드를 지원합니다. 방향키, 마우스 이동, 게임 입력 등에 사용할 수 있습니다.
로터리 엔코더 왼쪽 회전, 오른쪽 회전, 누르기 동작에 기능을 배치합니다. 볼륨, 브러시 크기 조절처럼 반복 입력이 필요한 작업에 적합합니다.
디스플레이 버튼 이미지, GIF/영상 썸네일, 텍스트 라벨, 현재 페이지 상태를 표시합니다.
RGB LED 버튼별 색상 또는 현재 페이지 전체 색상을 설정해 시인성을 높일 수 있습니다.
예시: 조이스틱은 마우스 이동, 로터리 엔코더는 볼륨 조절, 메인 키는 자주 쓰는 프로그램과 단축키로 구성하면 반복 작업이 빨라집니다.

3. PC Sync 모드

PC Sync 모드는 PC 프로그램이 실행된 상태에서 기기와 실시간으로 동기화되는 기본 사용 모드입니다.

아이콘과 애니메이션 설정

이미지 또는 영상 파일을 버튼 위로 드래그하면 PC 화면과 기기 디스플레이에 바로 적용됩니다.

사용 방법

  1. 파일 탐색기에서 JPG, PNG, GIF, WebP, MP4 등의 파일을 준비합니다.
  2. 원하는 버튼 위로 파일을 드래그 앤 드롭합니다.
  3. 삭제가 필요하면 오른쪽의 아이콘 지우기 또는 버튼 우클릭 메뉴를 사용합니다.
디스플레이에 텍스트 표시

버튼에 제목을 지정하면 기기 화면에 텍스트로 표시됩니다. 이미지가 있을 때는 이미지 위에 텍스트가 같이 보일 수 있습니다.

  1. 버튼을 선택합니다.
  2. 아래 설정 패널의 제목 입력칸에 표시할 문구를 입력합니다.
  3. 기능 변경 시 제목 자동 변경 옵션이 필요한 경우 체크합니다.
단축키 플러그인

자주 쓰는 키 조합을 버튼 하나로 실행합니다. 여러 키를 동시에 누르는 조합도 키보드에서 손을 뗐을 때 최종 조합으로 등록됩니다.

  1. 오른쪽 기능 목록에서 단축키 기능을 버튼에 드래그합니다.
  2. 아래 설정 패널에서 입력 칸을 클릭합니다.
  3. 예: Ctrl + Shift + S처럼 원하는 키 조합을 누른 뒤 손을 뗍니다.
로터리 엔코더 활용

로터리 엔코더의 좌회전, 우회전, 클릭에 각각 기능을 넣을 수 있습니다. 볼륨, 확대/축소, 브러시 크기 조절처럼 반복 입력에 유용합니다.

LED 색상 변경

선택한 버튼의 LED 또는 현재 페이지 전체 LED를 변경할 수 있습니다.

  1. 선택한 키 LED 변경 기능을 원하는 버튼에 드래그합니다.
  2. 색상 선택 창에서 LED 색을 선택합니다.
  3. 현재 페이지 LED 전체 변경을 사용하면 페이지 전체 색상을 한 번에 변경합니다.
액션 스플래시 이미지

기능이 실행될 때 화면에 원하는 이미지 또는 애니메이션을 잠깐 표시할 수 있습니다.

  1. 기능이 들어간 버튼을 선택합니다.
  2. 애니메이션 설정 기능을 해당 버튼에 적용합니다.
  3. 표시할 이미지와 재생 여부를 설정합니다.

4. 오프라인 모드

오프라인 모드는 PC 프로그램 없이 기기 내부 저장소의 설정으로 동작하는 모드입니다. 태블릿, 스마트폰, 아이패드 등에서 보조 키보드나 매크로 패드처럼 사용할 수 있습니다.

전환 지연: 오프라인 모드에 들어갈 때 기기 저장 데이터를 불러오므로 약간의 대기 시간이 생길 수 있습니다.
오프라인 이미지 저장

기기 내부 Flash에 이미지를 저장해 전원을 껐다 켜도 버튼 이미지가 유지되게 합니다.

  1. 상단의 오프라인 모드 버튼으로 오프라인 설정 화면에 들어갑니다.
  2. 이미지를 넣을 버튼을 선택합니다.
  3. 버튼 이미지 설정을 누르거나 파일을 버튼 위에 드래그 앤 드롭합니다.
  4. 기기에 저장을 눌러 설정을 전송합니다.
오프라인 제목과 텍스트

오프라인에서도 버튼 제목을 저장할 수 있습니다. 이미지가 없을 때는 텍스트 라벨로 표시됩니다.

오프라인 단축키, 매크로, 미디어 제어

단축키, 텍스트 매크로, 미디어 제어 기능을 기기에 저장해 모바일 환경에서도 사용할 수 있습니다.

단축키

키 조합을 저장해 앱 단축키를 실행합니다.

텍스트 매크로

자주 입력하는 문구를 한 번에 입력합니다.

미디어 제어

재생, 정지, 이전/다음 트랙, 볼륨 기능을 배치합니다.

오프라인 마우스 제어

마우스 클릭, 휠, 커서 이동 기능을 저장할 수 있습니다. 조이스틱과 함께 사용하면 태블릿이나 휴대폰에서도 커서 조작이 편해집니다.

오프라인 LED 저장

버튼별 LED 색상과 페이지별 분위기를 기기에 저장합니다.

5. 설정과 외관

설정 탭에서는 입력 방식, 화면 표시, 스킨, 언어, 사운드, 펌웨어 업데이트를 관리합니다.

조이스틱 모드
모드 설명
4방향 상, 하, 좌, 우만 인식합니다. 오입력을 줄이고 일반 단축키용으로 안정적입니다.
8방향 대각선 입력까지 허용합니다. 게임, 에뮬레이터, 이동 조작에 적합합니다.
아날로그 조이스틱의 기울기 값을 사용합니다. 아날로그 마우스 제어처럼 부드러운 입력에 사용할 수 있습니다.

연속 실행 옵션을 켜면 조이스틱이나 키를 누르고 있을 때 일정 간격으로 반복 입력됩니다.

스킨, 배경, 언어

스킨을 바꾸면 프로그램의 색상, 버튼 느낌, 배경 이미지, 일부 사운드가 바뀔 수 있습니다.

  • 스킨 선택: skins 폴더의 스킨을 선택합니다.
  • 배경 이미지 변경: 프로그램 또는 기기 화면 배경 이미지를 설정합니다.
  • 배경색 변경: 이미지가 없을 때 사용할 색상을 지정합니다.
  • 언어 설정: 한국어, 영어 등 지원 언어로 UI를 전환합니다.
팝업 디스플레이

기기 버튼을 누를 때 PC 화면에 현재 버튼 이미지와 기능을 팝업으로 표시합니다. 모니터 위치에 맞춰 표시 위치를 선택할 수 있습니다.

사운드와 시스템 트레이

마우스 호버 사운드와 드롭 사운드의 사용 여부와 볼륨을 조절할 수 있습니다. 창을 닫을 때 트레이에 남겨 계속 사용할 수도 있습니다.

6. 플러그인과 스토어

오른쪽 기능 목록에는 기본 도구와 플러그인 기능이 표시됩니다. 상단의 스토어 버튼을 통해 플러그인을 확인하고 확장할 수 있습니다.

기능 적용

오른쪽 기능을 원하는 버튼에 드래그하면 해당 버튼의 기능이 바뀝니다.

설정 패널

버튼을 클릭하면 아래쪽에 기능별 설정 화면이 나타납니다.

삭제와 초기화

기능 지우기, 아이콘 지우기, 애니메이션 설정으로 버튼 상태를 관리합니다.

7. 시스템 업데이트

소프트웨어 / 펌웨어 다운로드

최신 버전의 PC 제어 프로그램과 기기 펌웨어는 아래 링크에서 다운로드할 수 있습니다.

  • PC 프로그램: [다운로드 링크 준비중]
  • 기기 펌웨어: [다운로드 링크 준비중]

펌웨어 업데이트 방법

  1. 설정 탭에서 펌웨어 업데이트 버튼을 클릭합니다.
  2. 펌웨어 파일을 선택합니다.
  3. 안내에 따라 업데이트를 진행합니다.
  4. 업데이트 중에는 USB 케이블을 분리하지 않습니다.

8. 문제 해결

기기가 연결되지 않을 때
  • USB 케이블이 데이터 통신을 지원하는지 확인합니다.
  • 프로그램을 종료한 뒤 다시 실행합니다.
  • 다른 USB 포트에 연결해 봅니다.
이미지나 배경이 이상하게 보일 때
  • 이미지 파일이 너무 크면 적용 시간이 길어질 수 있습니다.
  • 오프라인 모드에서 배경이 없으면 현재 배경색 또는 기본 배경이 사용됩니다.
  • 이미지를 다시 지정한 뒤 기기에 저장합니다.
설정 패널이 늦게 뜰 때
  • 스킨 폰트나 WebView 폰트 로딩 문제가 있는지 확인합니다.
  • 스킨에 큰 폰트 파일을 넣을 수 있지만, WebView에는 폰트 파일을 직접 주입하지 않는 구성이 안정적입니다.
  • 문제가 계속되면 기본 스킨 또는 검증된 스킨으로 바꿔 비교합니다.
